## v1.1.1 - 2026-07-03

**Packaging + documentation release.**

- **Added `.pkgmeta`.** Without it, the BigWigsMods packager auto-generates the zip's `CHANGELOG.md` from raw git commit messages — which is what shipped in v1.0.0/v1.1.0. The packaged changelog now comes from the curated `docs/CHANGES.md` (same pattern as RGX-Framework), and `docs/` + `README.md` are excluded from the player zip.
- **README rewritten** around RGX-Hello's actual dual role: the canonical hello-world reference for addon developers, and RGX-Framework's in-game testing suite — including the `rgx-mcp` loop (the framework's MCP server can generate `core.lua`'s structure, and the framework's end-to-end test validates/audits this repo on every run), current test-tab coverage, and the not-yet-covered modules the suite should grow into.
- **Requires RGX-Framework v2.4.1+** (`X-RGX-Framework-MinVersion` bumped). The Volume slider's `%` suffix only renders from v2.4.1 — the framework's declarative options mapper silently dropped `suffix` before then. That bug was found by the framework's MCP test running against this very repo.
