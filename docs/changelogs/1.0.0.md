## v1.0.0 - 2026-07-03

**Initial release.**

RGX-Hello is the smallest complete RGX-Framework addon — a reference/template for building your own. The whole addon is one file (`data/core.lua`), built entirely from the `RGXAddon` declarative front door:

- Saved settings with automatic persistence
- A tabbed options panel with db-bound controls (toggle, slider)
- A slash command (`/rgxhello`) that opens the panel
- A minimap button
- Branded welcome message on login
- Scoped event registration through RGX-Framework's taint-safe paths

No event frames, no `C_Timer`, no `SLASH_X` globals, no SavedVariables boilerplate — RGX-Framework handles all of it.
