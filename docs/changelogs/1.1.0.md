## v1.1.0 - 2026-07-03

**RGX-Hello is now also RGX-Framework's visual QA harness.** Previously the QA harness lived as a separate in-tree dev tool inside the RGX-Framework repo (`tools/rgx-visual-test`), installed as its own addon. It's been merged into RGX-Hello instead — one repo, one install, two purposes:

- `data/core.lua` — the minimal declarative `RGXAddon` reference (unchanged).
- `data/visualtest.lua` — a hand-built options panel exercising every RGX-Framework control (color picker, toggles, sliders, nested dropdowns, font/texture dropdowns) via the manual API. New slash commands: `/rgxvisual` (`/rgxviz`) opens the full tabbed panel, `/rgxcolor` (`/rgxcp`) opens the color picker directly.

Also picked up a fix from RGX-Framework v2.4.0: the four "What to test" hint labels in the harness now wrap correctly instead of running past the panel edge (`UI:CreateLabel` gained an opt-in `width` option that enables word wrap).

**Requires RGX-Framework v2.4.0 or later.**
